# Bangladeshi Proxy Scraper

A scraper built on TypeScript and Bun. Scrapes from the following sources:

|Source|Method|
|---|---|
|proxydb.net|api|
|spys.one|puppeteer|

### To get started:

#### 1. Clone this repo

#### 2. Install Chrome for puppeteer

```bash
bunx @puppeteer/browsers install chrome
```

#### 3. Install dependencies:

```bash
bun i
```

#### 4. Run scraper

```bash
bun start
```
